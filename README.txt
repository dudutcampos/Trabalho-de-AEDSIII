# Trabalhode-AEDSIII
Trabalho desenvolvido durante a disciplina de Algoritmos e Estrutura de Dados 3 na UFSJ.

No repositorio contem a descrição do trabalho e a analize de desempenho dos algoritmos usados para simular uma pandemia com grafos.

Para copilar o programa foi acresentado um MakeFile e usa-se o comando: make
para rodar e so digitar o nome do arquivo executavel no terminal: ./tp

A entrada do programa e descrita no arquivo entrada.txt que se encontra na pasta raiz da aplicação, 
ela vem contendo as entradas encontradas nos testes do simulador e e por esse arquivo que se altera a entrada da aplicação.

Os relatorios completos utilizados para escrever o trabalho estao na pasta ResultadosObtidos junto com um bkp do arquivo de entrada.txt
